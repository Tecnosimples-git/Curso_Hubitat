# Capítulo 5: Dimmer MolSmart TCP

## Objetivo do Capítulo
Instruir a configuração de um módulo Dimmer MolSmart no Hubitat, reforçando os conceitos de controle de nível de brilho.

## Roteiro de Gravação

1. **Pré-requisitos de Rede**
   - Assim como o Relé, assumimos que o Dimmer já está operando na rede local.
   - Lembrar que a arquitetura de software é idêntica à do Relé (Pai e Filho) via rede local (TCP).
   - Tenha em mãos o Endereço IP do módulo Dimmer.

2. **Criando e Configurando o Pai**
   - Ir em **Devices -> Add Device -> Virtual**.
   - Criar com o Type `MolSmart Dimmers Parent`.
   - Configurar o Endereço IP em Preferences.
   - Salvar e verificar a criação dos canais (filhos).

3. **Testando e Controlando o Brilho**
   - Clicar no dispositivo filho recém-criado.
   - Mostrar que o driver filho agora é do tipo Dimmer (`MolSmart Child Dimmer`).
   - Diferente do relé, demonstrar o uso do comando "Set Level".
   - Inserir um valor (ex: 50) e clicar em Set Level.
   - Mostrar a alteração de estado no painel "Current States" (level: 50, switch: on).

4. **Rampas e Ajustes (Opcional)**
   - Citar brevemente se o driver oferecer ajustes de fade-rate / rampa, mostrando onde eles ficam nas preferências do dispositivo filho.

## Dicas para o Instrutor
- O processo é repetitivo de propósito. Reforce o conceito de que, após aprender a adicionar um MolSmart TCP, o padrão se aplica aos demais da linha LAN.
