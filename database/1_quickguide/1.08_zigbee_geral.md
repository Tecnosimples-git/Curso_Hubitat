# Capítulo 8: Zigbee Geral

## Objetivo do Capítulo
Abordar a malha Zigbee do Hubitat, enfatizando a importância do pareamento e como utilizar dispositivos homologados pela TRATO.

## Roteiro de Gravação

1. **Conceitos da Rede Zigbee**
   - Explicar rapidamente que o Hubitat possui uma antena Zigbee interna.
   - Diferente dos dispositivos TCP (que vão no roteador Wi-Fi), os Zigbee falam diretamente com o Hubitat
   - Zigbee usa diferentes canais, usamos canal 15,20,25 que são os que menos sofrem interferencia de wifi 
   - Zigbee deve ser adicionado na ordem do mais proximo para o mais distante para formar a rede mesh correta
   - explicar malha zigbee, que dispositivos podem ampliar o sinal usando uns aos outros como ponte.

2. **Validar o modelo do dispositivo e instalar o driver correto antes de adicionar o dispositivo zigbee ao módulo** 
   - usar tabela de referencia zigbee, 


2. **Colocando o Hub em Modo de Pareamento**
   - Navegar até **Devices -> Add Device -> Zigbee**.
   - Clicar em "Start Zigbee Pairing".
   - Mostrar o contador regressivo de 60 segundos na tela.

3. **Adicionando um Dispositivo na Prática**
   - Pegar um sensor ou relé Zigbee homologado TRATO.
   - Colocar o dispositivo físico em modo de pareamento (geralmente segurando o botão por 5 segundos).
   - Mostrar a tela do Hubitat localizando o dispositivo ("Found a Zigbee device...").
   - Nomear o dispositivo na tela.

4. **Verificando o Driver Correto**
   - Explicar que, ao contrário dos virtuais que criamos do zero, os Zigbee são detectados. Porém, o Hubitat pode escolher um driver genérico errado.
   - Consultar a tabela de drivers de referência Zigbee oficial da TRATO: [Planilha de Referência Zigbee](https://docs.google.com/spreadsheets/d/1haVMgZNGjW8X5_Dq8t_cTuD0xk04DoXrxlNVa_GZb0Y/edit?gid=0#gid=0).
   - Demonstrar como rolar até "Type", alterar manualmente para o driver específico homologado pela TRATO e clicar em "Save Device".
   - se for um relay, validar criação dos canais filhos e acionamento
   - se for keypad validar pressionar no botão nas variaveis state
   - se for sensor de contato testar contato. etc


## Dicas para o Instrutor
- Alerte os alunos que a troca manual de driver Zigbee geralmente exige clicar no botão "Configure" logo após a mudança para sincronizar os parâmetros. Demonstre isso!
